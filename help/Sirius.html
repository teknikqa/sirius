/Description/

Sirius - monitors network traffic on the chosen network interface and displays it continuously. It could be used for locating suspicious network traffic or to evaluate current utilization of the network interface. 

When I discovered Trafshow for Linux (Copyright (c) 1993-1997 JSC Rinet, Novosibirsk, Russia) I decided to create something like a Windows version of this usable software. Sirius was completely rewritten. It is not a Windows port of Trafshow. 

/Usage hints/

  1. It might take some time to display traffic information after starting network monitoring, when "Resolve addresses" is ON. If you switch it on the fly, it effects only on new connections. Please turn it ON only if you really need it. Once resolved Sirius is using its own resolve caching system. Addresses cache got lost when you close the program.
  2. When monitoring is started Sirius could decrease network performance of  your computer if there is too much network traffic to handle.
  3. If you are wondering how to get rid of About dialog, press <Esc> or <Alt> + <F4> or Right click or Double left click ;)
